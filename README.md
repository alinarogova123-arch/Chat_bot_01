# Work review notification bot

Бот телеграм для уведомлений о проверке работ.

## Запуск

Для запуска бота у вас уже должен быть установлен Python 3.

- Скачайте код
- Установите зависимости командой `pip install -r requirements.txt`
- Запустите бота командой `python main.py`

## Переменные окружения

Часть настроек проекта берётся из переменных окружения. Чтобы их определить, создайте файл `.env` рядом с `main.py` и запишите туда данные в таком формате: `ПЕРЕМЕННАЯ=значение`.

Доступны следущие переменные:
- `TELEGRAM_BOT_API_KEY` — Токен телеграм бота.
- `DEVMAN_API_TOKEN` — API Токен ученика сайта Devman.
- `TELEGRAM_CHAT_ID` — ID вашего чата с ботом.
